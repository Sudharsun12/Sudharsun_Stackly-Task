Student Registration Form is a responsive web application developed using HTML, CSS, and JavaScript. The form collects student details such as Full Name, Email, Phone Number, Course, Password, and Confirm Password. JavaScript is used to validate the input by checking for empty fields, verifying the email format, validating a 10-digit phone number, and ensuring that the password and confirm password fields match. Upon successful validation, the student's information is stored in the browser's Local Storage, a success message is displayed, and the form is reset. The project also includes a modern responsive UI with a Dark/Light Theme Toggle, providing a clean and user-friendly experience across different devices.
